## Instalación

Si queres instalar una versión de GeoCovid App podes hacerlo ejecutando
el siguiente código en R:

``` r
# install.packages("devtools")
devtools::install_github("Fernandez-Lab-WSU/geocovidapp")
geocovidapp::runapp()
```
